### 2. Cache::remember()
```php
$html = \Cache::remember('home_' . $url, 1440, function () use ($url) {
    if ($html = (new \HttpClient())->get($url)) {
        return $html;
    } else {
        throw new \Exception("Network Error, Try Later");
    }
});
```
